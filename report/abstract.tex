\chapter*{Abstract}%
%\addcontentsline{toc}{chapter}{\numberline{}Abstract}%
\addcontentsline{toc}{chapter}{Abstract}%

SecureEdgeNet is a Federated Learning based credit card fraud detection system that enables multiple simulated financial clients to collaboratively train fraud detection models without sharing raw transaction data. The project addresses the practical difficulty of fraud analytics, where transaction data is highly sensitive, geographically and institutionally distributed, severely imbalanced, and exposed to continuously changing adversarial behaviour. The proposed system uses Flower to coordinate federated training with Federated Averaging, where Logistic Regression and Multi-Layer Perceptron model parameters are trained locally on each client and aggregated centrally according to client sample counts. A client-local XGBoost layer is added to strengthen tabular pattern recognition, and the final fraud probability is produced through a configurable weighted ensemble of Logistic Regression, MLP, and averaged XGBoost probability scores. The pipeline includes automatic CSV loading, schema validation, missing value handling, scaling, categorical encoding support, stratified client partitioning, threshold tuning, model checkpointing, k-fold cross-validation support, and a lightweight FastAPI inference interface. Evaluation focuses on fraud-relevant metrics such as precision, recall, F1-score, ROC-AUC, PR-AUC, and confusion matrix analysis rather than accuracy alone, because fraud datasets are extremely imbalanced. The system demonstrates how privacy-preserving distributed intelligence and ensemble learning can be combined for realistic financial fraud detection research and educational experimentation.
